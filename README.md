* Smart wearable glove for sign language recognition
* Uses flex sensors to detect finger bending and hand gestures
* Built using Arduino Uno and embedded system technology
* Converts hand gestures into readable text in real time
* Displays output on a 16×2 LCD display
* Uses HC-05 Bluetooth Module for wireless communication with smartphones
* Portable, low-cost, and user-friendly design
* Helps deaf and mute individuals communicate easily
* Demonstrates sensor interfacing, analog signal processing, and gesture recognition
* Suitable for assistive communication and wearable technology applications
